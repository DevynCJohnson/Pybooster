def pybooster.basic.doeskeymvalue (   _dict)

Test if any key matches any dict value.

Return True if a key matches a value in the dict Examples doeskeymvalue(DICT)

  • { 'A': 'x', 'B': 'A', 'C': 'z'} => True
  • { 'A': 'A', 'B': 'y', 'C': 'z'} => True
  • { 1: 'x', 2: 'y', 'C': 1} => True
  • { 'A': 'x', 'B': 'y', 'C': 'z'} => False
  • { 'A': 'x', 'B': 'a', 'C': 'z'} => False Only int and str types are supported
  • { 'A': 'x', 'B': ['A', 'y'], 'C': 'z'} => False
