334 /**

336 
337 A floating-point number is composed of four elements:
338 
339  * SIGN: either negative or non-negative
340  * BASE (RADIX): which expresses the different numbers that can be represented with a single digit (2 for binary, 10 for decimal, 16 for hexadecimal, and so on)
341  * SIGNIFICAND (MANTISSA): which is a series of digits of the aforementioned base. The number of digits in this series is what is known as precision.
342  * EXPONENT (CHARACTERISTIC or SCALE): which represents the offset of the significand, affecting the value in the following way ->
343  * value of floating-point = SIGNIFICAND * (BASE ^ EXPONENT)
344 */
