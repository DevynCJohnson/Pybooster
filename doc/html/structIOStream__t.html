char(* peek) (uint16_t depth)

Inspect a byte in the stream's buffer without removing it.

Parameters
depth[in] Distance from the front of the read queue (i.e. the next byte to be read has depth 0, the byte after that has depth 1 and so on).
Returns
The byte value. In the case of an error (e.g. a byte does not exist at that depth) return an implementation-specific error code.

Definition at line 183 of file extra_datatypes.h.

uint16_t(* readLineTruncate) (char *buf, uint16_t maxLen)

Read the next available line from the stream but truncate the output. The implementation must define the line-ending sequence it uses to determine when a line is available. Line-ending characters are removed and the function outputs a NULL-terminated string. Where the output string would exceed the maximum specified length, truncate the string appropriately whilst still ensuring that the NULL-terminator is the last byte in the output.

Parameters
buffer[out] The target buffer.
maxLen[in] The maximum permissible length of the output (including the NULL-terminator) in bytes.
Returns
The length of the output string *not* including the NULL-terminator.
