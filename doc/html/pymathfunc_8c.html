DESCRIPTION

This is a Python3 module that uses functions from mathfunc.c Numerous mathematical and trigonometric functions. This extends the standard math library by providing by offering faster alternatives. Such functions include squareroot, fractorial, fib, haversin, isprime, ishexagonal, etc.
