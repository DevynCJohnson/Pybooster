double pka ( const double  ka)

Calculate the pKa from Ka (Acid Dissociation Constant)

Chemistry Explanation
[HA] = [H+] + [A-]; Molar concentrations
Ka = ([H+][A-]) / [HA]
Ka = ([X][0.01M+X]) / (0.02M-X); Ka = 10^-5M
X = 0.00001 * (0.02 / 0.01) = 2*(10^-5) = [H+]
pH = -log(2*(10^-5)) = 4.7

Parameters
[in]kaAcid Dissociation Constant

pka value
